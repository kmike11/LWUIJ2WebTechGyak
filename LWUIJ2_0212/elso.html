GitHub Parancsok

Helyi (Local) Műveletek:

  • git init - Új git repository inicializálása a helyi gépen.
  • git add <file> - Fájl hozzáadása a staging area-hoz.
  • git commit -m "Commit message" - Staged változások commitolása a repository-ba.
  • git status - Státusz lekérdezése a helyi repository-ról.
  • git log - Commitok listázása a repository-ban.

Távoli (Remote) Műveletek:

  • git clone <repository_url> - Egy távoli repository klónozása a helyi gépre.
  • git push <remote_name> <branch_name> - Lokális commitok feltöltése a távoli repository-ba.
  • git pull <remote_name> <branch_name> - Frissítés a lokális repository-ra a távoli repository-ból.
  • git remote add <remote_name> <repository_url> - Egy távoli repository hozzáadása mint távoli.
  • git remote -v - Az összes távoli repository listázása.
