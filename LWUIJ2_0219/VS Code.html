Visual Studio Code Beállítások és Bővítmények

Alapvető beállítások:

  • Témaválasztás: válassz egy témát a Beállításokban (Settings).
  • Betűtípus és méret beállítása.
  • Automatikus mentés bekapcsolása.
  • Nyelvi beállítások: tabulátorméret, nyelvi kódolás stb.

Bővítmények:

  • Prettier: Kódegyengető eKMöz.
  • ESLint: Kódstílus ellenőrző eKMöz JavaScript és TypeScript kódhoz.
  • GitLens: Git integráció bővítése a Visual Studio Code-ban.

Telepítsd ezeket a bővítményeket a Visual Studio Code Extensions fül alatt.
